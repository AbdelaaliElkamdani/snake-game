# Snake Game 🐍

A classic Snake Game built with **HTML, CSS, and JavaScript**.

## Features

* Control the snake using the keyboard arrow keys.
* Collect food to grow the snake and increase your score.
* Real-time score tracking.
* Game Over detection on wall or self-collision.
* Restart button to play again.
* Responsive and modern user interface.
* Multiple difficulty levels.

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* HTML5 Canvas

## How to Play

1. Open the game in your browser.
2. Click **Start Game**.
3. Use the arrow keys to move the snake.
4. Eat food to earn points and grow longer.
5. Avoid hitting the walls or the snake's own body.

## Installation

Simply download or clone the repository and open `index.html` in your browser.
